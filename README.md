# GuardaContactos
 
Hola, un proyecto para seguir mejorando mi front y backend, la idea es crear un sistema de creacion y logeo de usuarios, los cuales podran

1. Crear contactos 
1. Borrar Contactos
1. Actualizar Contactos
1. Ver sus contactos
1. Borrar la cuenta del usuario
1. Modificar informacion sobre el usuario

---
Las tecnologias a usar seran vue3, y Express, la base de datos usada sera postgresql y por ahora solo correra en localhost.
